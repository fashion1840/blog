# 道听书途 - 个人博客

> 一个基于 Hexo 的静态博客，分享科技、阅读与生活。

## 📝 每日科技摘要

本博客每天会自动更新一篇**每日科技摘要**，汇集过去24小时内最热门的技术动态。

### 内容来源
- **Hacker News** - 顶级技术社区的热门讨论
- **Reddit** - r/technology 和 r/programming 的最新帖子
- **Product Hunt** - 最新发布的科技产品

### 更新机制
- **时间**：每天早上 9:00（UTC+8）
- **自动化**：由 OpenClaw 自动搜集、整理并发布
- **结构**：按日期归档，路径格式 `YYYY/MM/DD/daily-digest/`

### 示例文章
- [2026-02-10 每日科技摘要](2026/02/10/daily-digest/)
